# Hospital-Management-System-cpp
A console-based hospital management system developed in C++ using object-oriented programming principles. 
The program manages doctors and patients through an interactive menu system and supports file-based data storage.
# Features
 - Add and manage patients
 - Add and manage doctors
 - Search patient by ID
 - Search doctor by ID
 - Display assigned doctors and patients
 - Count critical patients
 - Count in-patients
 - Find oldest patient
 - Persistent storage using text files
# Technologies Used
 - C++
 - Object-Oriented Programming (OOP)
 - STL Vectors
 - File Handling
 - Modular Design with Header and Source Files
 - Project Structure
 - doctor.h / doctor.cpp
 - patient.h / patient.cpp
 - hospital.h / hospital.cpp
 - driver.cpp
# Concepts Demonstrated
 - Classes and Objects
 - Constructors
 - Encapsulation
 - Dynamic memory allocation
 - Vectors
 - File I/O
 - Menu-driven applications
